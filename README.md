# project : Credit Card Fraud Detection using Machine Learning  

## Introduction  
Developed a fraud detection system using the Credit Card Fraud Detection dataset (Kaggle) to identify fraudulent financial transactions. Performed data exploration, preprocessing, and handled severe class imbalance using techniques such as Random Undersampling, Oversampling, and SMOTE. Implemented and compared several Machine Learning models including Logistic Regression, KNN, SVC, Decision Tree, Random Forest, and an Artificial Neural Network (ANN). Evaluated model performance using precision, recall, F1-score, ROC curves, and confusion matrices to identify the most effective approach for detecting rare fraudulent transactions.
